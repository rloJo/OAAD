## 문제 조건
> <a href = "https://www.acmicpc.net/problem/11444"> 문제 링크 </a>  
> 시간제한 : 1초  
> 메모리 제한 : 256MB  
> 정답 비율 : 47.508%

### 풀이 과정
---
1. N이 너무 커서 dp방법으로 푸는 것은 불가능하다.
2. 따라서 행렬을 이용한 풀이를 해야한다.
3. 또한 분할정복을 이용해 행렬의 제곱 계산을 빠르게 해야한다 $O(N)$
